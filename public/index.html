Google Photos Cleaner

Elimina masivamente tus fotos y videos de Google Fotos

Advertencia: Las fotos eliminadas no se pueden recuperar despues de vaciar la papelera. Antes de continuar, haz un respaldo en Google Takeout.
PASO 1

Abre Google Fotos en una nueva pestana

Asegurate de estar con la cuenta de la que quieres eliminar las fotos.

Abrir Google Fotos
PASO 2

Abre la consola del navegador

En Chrome o Edge: presiona F12 y luego click en la pestana Console

En Mac: presiona Cmd + Option + J

En Firefox: presiona F12 y click en Consola

PASO 3

Copia el script y pegalo en la consola

Haz click en el boton de copiar, luego pega con Ctrl+V o Cmd+V en la consola y presiona Enter.

(function() {
  async function sleep(ms) {
    return new Promise(r => setTimeout(r, ms));
  }
  async function run() {
    const photos = document.querySelectorAll('div[data-latest-bg]');
    if (!photos.length) {
      alert('No se encontraron fotos. Asegurate de estar en photos.google.com');
      return;
    }
    photos[0].dispatchEvent(new MouseEvent('mouseover', {bubbles: true}));
    await sleep(300);
    photos[0].click();
    await sleep(300);
    photos.forEach(p => p.click());
    await sleep(500);
    const moreBtn = document.querySelector('button[aria-label="More options"]') ||
                    document.querySelector('button[aria-label="Mas opciones"]');
    if (moreBtn) moreBtn.click();
    await sleep(500);
    const deleteBtn = Array.from(document.querySelectorAll('li[role="menuitem"]'))
      .find(el => el.textContent.includes('Delete') ||
                  el.textContent.includes('Eliminar') ||
                  el.textContent.includes('papelera'));
    if (deleteBtn) {
      deleteBtn.click();
      await sleep(500);
      const confirmBtn = document.querySelector('button[data-mdc-dialog-action="ok"]') ||
        Array.from(document.querySelectorAll('button'))
          .find(b => b.textContent.includes('Move to trash') ||
                     b.textContent.includes('Mover a la papelera'));
      if (confirmBtn) {
        confirmBtn.click();
        await sleep(1000);
        alert('Lote movido a la papelera. Recarga Google Fotos con F5 y ejecuta el script de nuevo.');
      } else {
        alert('No se encontro el boton de confirmar. Intentalo de nuevo.');
      }
    } else {
      alert('No se encontro el boton de eliminar. Intentalo de nuevo.');
    }
  }
  run();
})();
PASO 4

Repite hasta eliminar todo

El script mueve un lote a la papelera cada vez. Recarga Google Fotos con F5 y vuelve a ejecutar el script hasta que no queden fotos.

Informacion importante

  • Las fotos van a la papelera, no se eliminan de inmediato
  • Tienes 60 dias para recuperarlas desde la papelera
  • Para eliminar definitivamente: ve a Papelera y haz click en "Vaciar papelera"
  • Si tienes miles de fotos, necesitaras ejecutar el script varias veces
Esta herramienta funciona completamente en tu navegador. Ningun dato es enviado a ningun servidor.
